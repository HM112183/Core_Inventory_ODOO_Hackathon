# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
   `npm install tsx --save-dev`
2. Run the app:
   `npm run dev`

# Features available according to the User login -
   1. Admin: All permission granted
   2. Manager: All allowed except Infasturcture mangement
   3. Staff: Adjustments and Move history permission not given
